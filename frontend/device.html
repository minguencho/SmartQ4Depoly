Read before Regist


You have to follow below commands in your device bash

  • 1. git clone https://github.com/BangMingGing/SmartQ-Device.git
  • 2. cd SmartQ-Device
  • 3. pip3 install -r requirements.txt
  • 4. python3 device.py --smartq_id={Your SmartQ ID} --device_name={Your Device Name}
  • 5. You can register your {Your Device name} down here
